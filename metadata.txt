about=ArcheoGlyph helps archaeologists create factual, standardized symbols for GIS maps. Upload artifact or feature images and generate map-ready symbols using Auto Trace, AI (Hugging Face, Google Gemini, or local Stable Diffusion), or built-in templates. Features include style controls (Colored/Typology/Line/Measured), optional SAM segmentation, target layer selection, color adjustment, proportional sizing, and direct QGIS symbol library integration.
