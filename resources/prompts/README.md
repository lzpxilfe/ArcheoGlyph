# AI Prompt Templates

This directory contains prompt templates for AI-based symbol generation.

## File Format

Each `.txt` file contains a prompt template for a specific style:
- cute_kawaii.txt - Prompts for cute/kawaii style
- minimal.txt - Prompts for minimalist style 
- classic.txt - Prompts for classic archaeological style

## Variables

Templates can use placeholders:
- {artifact_type} - Type of artifact
- {color} - Target color
- {style_details} - Additional style details
